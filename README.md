# 📊 Target E-Commerce Sales Analysis using SQL (BigQuery)

## Project Overview

This project analyzes the Target Brazil E-Commerce dataset using Google BigQuery SQL. The objective is to extract meaningful business insights related to customer behavior, order trends, sales performance, payment methods, freight costs, and delivery performance.

The project demonstrates practical SQL skills required for Data Analyst roles, including joins, aggregations, window functions, Common Table Expressions (CTEs), date functions, and business-oriented analysis.

---

## Business Problem

Target operates one of the largest e-commerce platforms. Understanding customer purchasing patterns, delivery efficiency, and payment behavior can help improve operational performance and customer satisfaction.

This project answers several business questions through SQL analysis.

# Key Insights

- Orders increased significantly between 2017 and 2018.
- Most purchases occurred during daytime hours.
- Certain states generated considerably higher sales revenue.
- Freight costs varied significantly across regions.
- Credit cards were the most frequently used payment method.
- Delivery performance differed across Brazilian states.
